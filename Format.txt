Formatted input output

The format in fortran are of eight types:
    ~I format 
        --> Integer Format
        iw.m
        w>=m
        For Example: 
            -13 has three width
            1234 has width 4
            -7 has width 2
        FORMAT(I3,I4,I2)

    ~X format
        -->Skip Format --skip columns
        2x -->Skip 2 columns
        3x -->Skip 3 columns
        FORMAT(I4,2x,I3,3x,I2)

    ~F format
        -->For floating number
        Fw.d
        w > d
        For Example:
            1.2346===FORMAT(f6.4)
    ~E format
        -->Exponent Format 
        Ew.d 
        w >= 7 + d
        For Example:
            1.2E12 === FORMAT(E6.2)

    ~/ format
        Same as \n in C Programing language
        // two line skip

    ~A format
        Generally used for character
        Aw
        Character x*10
        X = Hello_World
        FORMAT(A5)
        Only hello gets display

    ~T format
        Tab Format
        TW 
        FORMAT(I4,T6,I2,2x,I3)

    ~quote
        If we want to display Pulchowk
        FORMAT('PULCHOWK')
